# akurkar07.github.io

Personal website for Alex Kurkar, hosted with GitHub Pages.

The site is a small portfolio for my projects, hackathons, programming interests, and background as a Computer Science student. It is intentionally static and lightweight so it stays easy to maintain.

## Structure

```text
.
+-- index.html
+-- about.html
+-- assets/
    +-- css/
    |   +-- site.css
    +-- js/
        +-- site.js
```

- `index.html` is the homepage and timeline.
- `about.html` is the about page.
- `assets/css/site.css` contains the shared styling.
- `assets/js/site.js` contains the shared JavaScript.

## Planned Additions

I plan to add interactive demos for some of my projects, especially around interpreters, compilers, language design, and low-level systems.
